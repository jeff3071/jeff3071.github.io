從Firebase取得資料

前面有介紹到如何連接Firebase，而連接的程式碼就放在app.py裡面

為了不讓app.py太過雜亂，所以我把取資料的程式碼獨立為db_connect.py，並且連同之前儲存資料的程式碼一起放進來。

取得資料的方法非常簡單，指定好collection和document後，使用get()就能取得，取出來的資料可以用to_dict()轉成dict，讓Python好處理。
